## 配置文件

| ---------------- | ----------------------------------------- | --------------------------- |

| keybindings.json | `C:/Users/User/AppData/Roaming/Code/User` | Visual Studio Code 键位配置 |
| extensions.txt   |                                           | Visual Studio Code 插件列表 |
