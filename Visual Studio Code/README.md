## 配置文件

| ---------------- | ------------------------------------------ | -------------------------------- |
| snippets         | `C:/Users/User/AppData/Roaming/Code/User/` | Visual Studio Code Snippets 文件 |
| settings.json    | `C:/Users/User/AppData/Roaming/Code/User/` | Visual Studio Code 全局配置文件  |
| keybindings.json | `C:/Users/User/AppData/Roaming/Code/User/` | Visual Studio Code 键位配置文件  |
| extensions.txt   |                                            | Visual Studio Code 插件列表      |
