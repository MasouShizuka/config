| cn_dicts                 | `$APPDATA/Rime/` | 词库目录             |

| weasel.yaml              | `$APPDATA/Rime/` | 小狼毫的前端配置文件 |
| *.custom.yaml            | `$APPDATA/Rime/` | 自定义文件           |
| *.dict.yaml              | `$APPDATA/Rime/` | 词库                 |
| *.schema.yaml            | `$APPDATA/Rime/` | 输入法方案           |
| custom_phrase.txt        | `$APPDATA/Rime/` | 自定义短语           |
