# starship

## 仓库

- [https://github.com/starship/starship](https://github.com/starship/starship)

## 配置文件

| 文件          | 路径                      | 说明     |
| ------------- | ------------------------- | -------- |
| starship.toml | `$HOME/.config/starship/` | 配置文件 |

## 注意

`Windows` 端默认的安装路径为 `C:/Program Files/starship/bin/starship.exe`，而若路径中包含 `space` 则无法执行 `starship init` 命令，因此需要更改安装路径为不含 `space` 的路径
