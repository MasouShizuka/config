# KeyMouse

## 仓库

- [https://github.com/iscooool/KeyMouse](https://github.com/iscooool/KeyMouse)

## 配置文件

| 文件        | 路径        | 说明                  |
| ----------- | ----------- | --------------------- |
| KeyMouse    | `KeyMouse/` | KeyMouse 修改后的源码 |
| config.json | `KeyMouse/` | KeyMouse 配置文件     |

## 注意

- 本人对 `KeyMouse` 项目进行过修改，源码位于 `KeyMouse` 文件夹内，`config.json` 需要配合修改后的源码编译出的 `KeyMouse.exe`
    - 编译修改后的源码：
        1. 运行命令 `git clone --recurse-submodules https://github.com/iscooool/KeyMouse.git`
        2. 将修改后的源码覆盖对应的文件
        3. 安装 `Visual Studio` 并导入项目
        4. 以 `x64` 的方式编译项目
