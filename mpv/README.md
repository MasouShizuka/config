## 配置文件

| 文件        | 路径                   | 说明           |
| ----------- | ---------------------- | -------------- |
| fonts       | `mpv/portable_config/` | 字体文件       |
| scripts     | `mpv/portable_config/` | 外置脚本       |
| script-opts | `mpv/portable_config/` | 外置脚本配置   |
| shaders     | `mpv/portable_config/` | 滤镜           |
| vapoursynth | `mpv/portable_config/` | 补帧           |
| input.conf  | `mpv/portable_config/` | 快捷键配置文件 |
| mpv.conf    | `mpv/portable_config/` | 全局配置文件   |
